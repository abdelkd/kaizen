# Korean - Letters - ㅏ

**Name:** ( a )
**Type:** Basic vowel
**Sound:** /a/ in *car* or *father*
**Stroke order:** draw vertical line down first then a short horizontal line to the right.

**Usage:** always used next to the consonant in a syllable block, for example:

- **가** (ga/ka) = ㄱ (g/k) + ㅏ (a)
- **나** (na) = ㄴ (n) + ㅏ (a)
